# Cell Growth Simulator

A Streamlit app that models cell population using:

- Exponential growth/decay
- Logistic growth with carrying capacity (K)

## Features
- User-controlled parameters (N, r, d, steps, K)
- Real-time simulation
- Graph with:
  - Cell population
  - Carrying capacity
  - Equilibrium point

## How to run
pip install -r requirements.txt
streamlit run app.py
